# Group Homework Submissions – Class 7.5

This directory contains all collaborative homework submissions for Class 7.5 in the TheoWAF program.
Assignments in this section reflect coordinated group work, shared responsibility, and collective problem‑solving across the cohort.

---

## Purpose of This Directory

The Group Homework section is intended to:

- Document team‑based assignments completed throughout the course
- Demonstrate collaborative analysis, design, and implementation skills
- Provide instructors with a clear, organized structure for grading group deliverables
- Maintain a consistent record of shared work products across the class

---

## Expectations for Group Work

Group submissions should:

- Reflect contributions from all assigned team members
- Follow the technical and formatting requirements defined in each assignment
- Include clear explanations, diagrams, or code samples where appropriate
- Demonstrate accuracy, completeness, and reproducibility

Where applicable, teams should also include:

- A brief summary of the division of responsibilities
- Notes on challenges encountered and how they were resolved
- Any assumptions made during the assignment

---

## Directory Layout

Each assignment should be placed in its own subfolder:

```
Group-Homework/
├── 01-assignment-name/
├── 02-assignment-name/
└── ...
```

Each subfolder should contain:

- A `README.md` describing the task and approach
- Any required artifacts (code, diagrams, screenshots, documentation)

---

## Academic Integrity

All submissions must adhere to course policies regarding collaboration, originality, and citation of external sources.
This directory contains only original work produced by the assigned group members for educational purposes.
