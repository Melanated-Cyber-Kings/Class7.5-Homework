# Individual Homework Submissions – Class 7.5

This directory contains my individual homework submissions for Class 7.5 in the TheoWAF program.
Assignments in this section demonstrate my personal understanding, technical proficiency, and mastery of course concepts.

---

## Purpose of This Directory

The Individual Homework section is intended to:

- Provide instructors with a clear, organized record of my personal work
- Demonstrate comprehension of weekly topics and hands‑on exercises
- Track my progress across the duration of the course
- Reinforce accuracy, reproducibility, and professional documentation practices

---

## Structure and Organization

Each assignment is stored in its own subfolder:

```
Individual-Homework/
├── 01-week-01/
│   └── README.md
├── 02-week-02/
│   └── README.md
└── 03-week-03/
└── README.md
```

Each assignment folder includes:

- A `README.md` summarizing the task, methodology, and results
- Any required supporting files (scripts, configs, screenshots, notes)

---

## Submission Standards

Individual submissions should:

- Be written clearly and professionally
- Follow all assignment instructions and formatting requirements
- Include explanations of reasoning, not just final answers
- Demonstrate reproducible steps where technical tasks are involved
